## [v0.1.0] - 2026-02-28

### Highlights

- Go accessibility auditing toolkit for WCAG 2.0, 2.1, and 2.2 compliance testing
- axe-core integration with LLM-as-a-Judge evaluation to reduce false positives
- Multiple audit modes: single page, site crawling, and user journey testing
- Report formats: JSON, HTML, Markdown, VPAT 2.4, WCAG-EM, CSV

### Added

- Core `Auditor` type with `AuditPage()`, `AuditSite()`, and `AuditJourney()` methods ([`9c090af`](https://github.com/plexusone/agent-a11y/commit/9c090af))
